Todos los ciudadanos de Los Santos deben aprobar su examen antes de poder conducir.

Test Teorico
- La prueba de teoría cuesta 500$, esto no se reembolsa si no pasa la prueba.
- No tenga miedo, la escuela de manejo acepta crédito, pero tenga cuidado de no endeudarse.
- Si no aprueba la prueba la primera vez, no puede volver a tomarla inmediatamente, tendrá que tomarla en una fecha posterior.

Test de Conduccion
- El examen práctico cuesta 2500$, al igual que el examen teórico, este pago no se reembolsará si no pasa la prueba.
- Asegúrate de estar alerta mientras conduces y evita accidentes!
